# Gym Management System Database 🏋️‍♂️

Энэхүү төсөл нь Фитнесс клубын мэдээллийн системийн өгөгдлийн сангийн бүтэц болон аюулгүй байдлын тохиргоог агуулсан.

## Бүтэц:
- **Members**: Гишүүдийн мэдээлэл
- **Trainers**: Дасгалжуулагчдын мэдээлэл
- **Subscriptions**: Гишүүнчлэлийн бүртгэл
- **Attendance**: Ирцийн бүртгэл

## Хэрхэн ажиллуулах вэ?
1. MySQL Workbench эсвэл Terminal нээнэ.
2. `mysqlScript.sql` файлыг ажиллуулна.
3. Хэрэглэгчийн эрхүүд болон Query-нүүд автоматаар үүсэх болно.
